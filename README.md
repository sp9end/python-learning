# Python Learning Journey 🐍

A structured Python learning path from scratch, focused on data analysis skills.
No external libraries — only Python built-ins.

## Topics Covered

- Variables, data types, operators
- Data structures: lists, dictionaries, tuples, sets
- Loops: for, while, list comprehensions
- Functions, *args, **kwargs
- File handling: CSV and JSON
- Built-in modules: csv, json, os, datetime, collections, statistics
- Object-Oriented Programming: classes, methods
- Error handling: try/except

## Projects & Exercises

Each topic includes practical exercises with real-world data.
